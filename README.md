This is supposed to become an alternative to [HomeBank](https://www.gethomebank.org/), because I am frustrated with it.
HomeBank seems to be a great app, but it also seems more complicated than necissary.

MyBudget tries to solve this by:
- Being a web-hosted service -> Access from anywhere at all times
- Less is more -> Less features, but enough for simple overview of balances accross bank accounts, budgeting and saving goals
